Netty 3.6.6.Final released

by normanmaurer
on 15-May-2013

One month after the release of Netty 3.6.5.Final it was about time to ship 3.6.6.Final. This release is cut from the 3.x branch and is a bug-fix only release, so no new features are included. You can get it from the downloads page and the Maven central repository. Please report any regressions or bugs via the issue tracker.

In total this release "fixes" 16 isses.

Noteworthy changes

  • Fix multiple issues when handling HTTP multipart. (#1228), (#1313), (#1332) and (#1358)
  • HttpChunkAggregator throws IllegalArgumentException because of incorrect removal of chunked transfer-encoding (#1322)
  • Deadlock can happen when MemoryAwareThreadPoolExecutor with limit is used (#1310)

Visit here for the complete list of the changes.
