Say 'Hello' to Netty 3.5.3.Final

by normanmaurer
on 23-Jul-2012

Only a few weeks after the last release, the Netty project team is proud to announce another bugfix release out of the 3.5.x Series. Netty 3.5.3.Final comes with many bugfixes, so please consider upgrading as soon as possible.

Check out our issue tracker for a full description of all of them.

The JAR file can be found on Maven Central as usually. Beside this you can also get the full tarball from the download page.

Please let us know if you find any issues, in the meantime have fun develop via Netty.

Enjoy!
