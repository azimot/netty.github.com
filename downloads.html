Dependencies

Please note that Netty has no mandatory external dependencies. JDK 1.5 (1.6 for Netty 4.x) or above is all that you need to run Netty.

Getting notified on a new release

You can either subscribe to our official blog or follow @netty_project at Twitter.

Downloading from the Maven central repository

Add the following dependency section to your pom.xml:

<dependencies>
  ...
  <dependency>
    <groupId>io.netty</groupId>
    <artifactId>netty</artifactId>
    <version>X.Y.Z.Q</version>
    <scope>compile</scope>
  </dependency>
  ...
</dependencies>
